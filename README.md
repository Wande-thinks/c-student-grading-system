# Student Grading System

A simple C program that accepts student names and scores in Mathematics, English, and Physics, calculates the average score, assigns a grade, and displays the results.

## Features

- Accepts student information
- Calculates average scores
- Assigns grades based on average performance
- Validates score inputs (0–100)
- Processes multiple students using a loop

## Grading Scale

| Average Score | Grade |
|--------------|--------|
| 70 and above | A |
| 60 – 69 | B |
| 50 – 59 | C |
| 45 – 49 | D |
| Below 45 | F |

## Technologies Used

- C Programming Language
- GCC Compiler

## How to Compile

```bash
gcc student_grading_system.c -o student_grading_system
```

## How to Run

```bash
./student_grading_system
```

# Student Grading System

A C program that accepts student scores in Mathematics, English, and Physics, calculates the average score, and assigns a grade.

## Features

- Accepts student names
- Calculates average scores
- Assigns grades (A-F)
- Validates scores (0-100)
- Processes multiple students

## Compilation

gcc student_grading_system.c -o student_grading_system

## Run

./student_grading_system
